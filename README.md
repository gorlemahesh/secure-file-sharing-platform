# Secure File Sharing Platform

A cloud-based platform that allows authenticated users to upload files securely and generate temporary download links.

## Architecture

Frontend → API Gateway → Lambda → S3 → DynamoDB

## Tech Stack

- AWS Lambda
- Amazon S3
- Amazon DynamoDB
- Amazon Cognito
- API Gateway
- React (Frontend)

## Features

- Secure file upload
- Temporary download links
- Authentication using Cognito
- File metadata tracking
